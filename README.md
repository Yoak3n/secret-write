### Usage
Alt + Q 显示/隐藏窗口  
Alt + A 显示/隐藏边框（防止找不到在哪以及支持拖动）  
Alt + Z 打开设置子窗口   
1. 字体颜色  
2. 输入框背景颜色  
3. 输出文件路径（默认在安装目录下的`output.txt`)
